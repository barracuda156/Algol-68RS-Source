Ex 4.7
The second operand of OREL is only elaborated if the first yields FALSE. 
   PROGRAM ex4 7 CONTEXT VOID
   USE standard
   BEGIN
      INT a = 3, b = 5, c = 4;

      IF
         IF a > b
         THEN TRUE
         ELSE b > c
         FI
      THEN print("Ok")
      ELSE print("Wrong")
      FI
   END
   FINISH
